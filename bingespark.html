Binge Spark Movie Review Website

Description.

This project was a web-based movie review application that was completed as part of the Web Development module for MSc Sofware Development. This required the use of HTML, JavaScript, PHP, and SQL. This website was hosted using a LAMP stack, allowing for the website to be easily accessed online and for the website data to be stored on a MySQL database. A REST API was also created and performed all interactions with the database. The movie review website allowed its users to search for movies and filter using different criteria, such as: genre, director, most-liked, ranking, actors, runtime, etc. Registered users could add reviews, favourites, and rate movies. Administrators had all the functionality of a user but could also edit all data related to movies on the website.
